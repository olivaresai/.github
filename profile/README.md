**Ground truth for enterprise AI.**

Know what your AI touches, decide what it can do, and prove it.

</div>

Olivares AI is an open, self-hostable product to **discover, operate and govern the AI agents already running on your infrastructure** — every agent, session, model and MCP server, across Linux, Docker, Kubernetes, cloud and air-gapped estates. Its pillar is a read/write **access map** of what each agent can reach, with a permitted-vs-observed diff that surfaces least-privilege drift. Self-hosted and vendor-neutral: your governance data never leaves your perimeter.

- **[olivares](https://github.com/olivaresai/olivares)** — the product. One static Go binary with an embedded console; AGPL-3.0 core with an Apache-2.0 connector SDK (open core, no feature-capping).
- **[olivares.ai](https://olivares.ai)** — website and documentation.
- **Security:** report vulnerabilities privately per [SECURITY.md](https://github.com/olivaresai/olivares/blob/main/SECURITY.md) — never via a public issue.
- **Commercial licensing:** `enterprise@olivares.ai` ([details](https://github.com/olivaresai/olivares/blob/main/LICENSING.md)).

Also from Olivares.AI: **[Alma](https://alma.olivares.ai)** — a SaaS AI workspace (chat, image, video, audio and creative studios) built on a persistent, visual, manageable memory system.
